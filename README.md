# Le Pong de Franciscus

Mini-jeu HTML5 rétro en local : Pong football, terminal MS-DOS, cages phosphore et dramaturgie douteuse.

## Accès

Le jeu affiche un écran d'accès. Le mot de passe accepté correspond au prénom d'un joueur du dossier `assets/images`, sans tenir compte des majuscules ni des accents.

Exemples :
- `Francisco`
- `Julien`
- `Sandra`
- `Hubert`
- `Louis-G`
- `Olivier`
- `Patricia`
- `Kamar`
- `Yohan`
- `Vijay`
- `Audrey`
- `Alexis`
- `Mostapha`
- `Louis-L`
- `Stéphanie`
- `Lauriane`
- `Cécile`
- `Florence`

Le mot de passe caché `WARGAME` reste disponible si `ENABLE_WARGAME` vaut `true` dans `js/targets.js`.

## Local

Ouvrir `index.html` dans un navigateur moderne, ou lancer `lancer-jeu.command` sur Mac.
